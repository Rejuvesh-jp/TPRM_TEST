  • Duplicate gaps (same root cause, different wording)
  • Unsupported gaps (no questionnaire / artifact backing)
  • Severity misalignments
  • Vague recommendation wording

{# ── Version Report Cache ── #}
Version Report Cache
Whole-pipeline fingerprint bypass
{% if user and user.role == 'admin' %}
{% else %} {% if llm_cache_enabled %}ON{% else %}OFF{% endif %} {% endif %}

Fingerprints the entire assessment input (file bytes, evidence chunk IDs, prompt versions, model, judge state). If the fingerprint matches a previous completed assessment for the same vendor, the entire pipeline is bypassed and that report is returned instantly — no LLM calls, no embeddings, no OCR.
